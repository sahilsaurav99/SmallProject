/* DICE GAME */

RULES or How to PLAY :
1. You'll get $1000 default
2. The computer will ask you how much money you want to bet
3. You'll put the bet then It will ask you to guess between 1-6
4. If the guess will right you'll Win the money
5. If you guessed wrong you lost the bet money
6. You can continue to the next round.
6. If you want to quit the game just choose 0.
